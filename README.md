🌌 # Marvel Fan Portal

An interactive and eye-catching fan portal for Marvel enthusiasts.
This project brings the Marvel Universe to life with smooth animations, hero sections, and engaging UI.

🚀 # Features

🎬 Landing Animation – Smooth scroll and swipe-in animations inspired by Figma design.

🦸 Hero Sections – Each Marvel character (Iron Man, Thor, Captain America, etc.) has a dedicated section with background effects.

✨ Hover & Click Effects – Hover reveals hero traits, and clicking opens a mini modal form (to share why you love the hero).

🎨 Custom Fonts & Styling – Designed with gradients, imported fonts (like Avengence), and themed backgrounds.

📱 Responsive Design – Works seamlessly across devices.

🛠️# Tech Stack

HTML5 – Structure of the portal

CSS3 – Styling, gradients, custom fonts, and animations

JavaScript (Vanilla JS) – Scroll animations, modal handling, interactivity
